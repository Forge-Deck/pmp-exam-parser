PMI Exam Analysis Parser

Paste the source from your PMI Exam Analysis page and this will pull out the buried task scores. Full page source, body-only HTML, the Kendo chart line, or the raw JSON block should all work as long as the task data is still there. Everything runs in your browser.

Paste your source

Paste the source below, or drop/upload a .html or .txt file.

Getting the source

  1. Open PMI CCRS → Report PDUs → Exam Analysis.
  2. Open your exam result.
  3. Right-click → View Page Source, or use F12 → Sources.
  4. Copy the source text and paste it here.

The useful numbers are hidden in the chart data, not in the visible table. This page looks for ScorePercentage, TotalQuestions, and Tasks, then turns those into estimated raw scores.

Names, PMI IDs, links, analytics, and account fields are ignored and are not included in the report.

Scoring note: this is a raw-score estimate. PMI's official pass/fail and AT/T/BT/NI ratings may use scaled scoring, so the bands may not match fixed raw percentages.

Band estimate: the band section is based on a small set of community datapoints and older/pre-update reports. It is a rough guide only and has not yet been proven against the new PMBOK8-oriented results.
